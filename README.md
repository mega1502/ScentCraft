# مشروع تخرج: نظام تركيب العطور الافتراضي (Perfume Virtual Lab) 🧪🌸

## 📝 وصف المشروع
مشروع مبتكر يهدف إلى تحويل عملية شراء العطور إلى تجربة تفاعلية. يتيح الموقع للمستخدمين تصميم عطورهم الخاصة من خلال "مختبر افتراضي" واختيار المكونات والنسب التي تناسب أذواقهم، مع إمكانية طلب العطر المصمم ليصلهم حتى باب البيت.

## ✨ المميزات الرئيسية
* **المختبر الافتراضي (Virtual Lab):** واجهة تفاعلية لاختيار المكونات العليا، الوسطى، والقاعدية للعطر.
* **نظام النسب:** التحكم في تركيز كل مكون داخل الزجاجة.
* **كتالوج العطور:** عرض لعطور جاهزة مع إمكانية الفلترة حسب النوع.
* **عربة التسوق:** نظام متكامل لإضافة المنتجات وإتمام عملية الشراء.
* **لوحة تحكم (لو كانت موجودة):** لإدارة المنتجات والطلبات.

## 🛠️ التقنيات المستخدمة

* **Backend:** (اكتب اللغة المستخدمة مثل PHP, Node.js, or Laravel)
